Trabalho sobre DOM

1. O que é DOM?

O DOM (Document Object Model) é uma representação do HTML em forma de estrutura de árvore. Ele permite que o JavaScript acesse e modifique os elementos da página. Ou seja, o HTML cria a estrutura e o DOM permite manipular essa estrutura.

2. O que é um nó (node)?

Um nó é qualquer parte do documento HTML. Existem três tipos principais: elementos (como <p> e <div>), atributos (como id e class) e textos (o conteúdo dentro das tags).

3. Métodos de seleção

getElementById(): seleciona um elemento pelo id.
querySelector(): seleciona o primeiro elemento usando seletor CSS.
querySelectorAll(): seleciona todos os elementos correspondentes.

4. innerHTML vs innerText

innerHTML permite inserir HTML dentro do elemento.
innerText altera apenas o texto.
A diferença é que o innerHTML interpreta tags HTML.

5. Eventos

Eventos são ações do usuário na página.
Exemplos:
onclick: quando clica
onchange: quando muda algo
onmouseover: quando passa o mouse
